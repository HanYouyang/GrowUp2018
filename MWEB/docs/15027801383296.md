# 房产策划转行公关的经验
转发者评：作者在房产策划工作两年后，转行进入顶尖的公关公司。作者给出了他转行找工作的心得和经验，我觉得这些宝贵的经验对于任何想转行的人都很值得参考。

2014年来北京的第一周就把工作落实了，进入了理想行业全球前十的一家公司。offer拿的比想象的顺利，如果真正算起来，找工作花的时间其实有一个月左右。从一开始的海投到后来有方向的投，其实每个阶段都有不顺、有疑惑，还好走过来了，心态也还算平和，始终保持一种比较自信的姿态。

现在仍能想起某家的面试，还没来得及回答几个问题就被拒绝了，也是第一次那么直接了当的被拒绝。无外乎没专业工作经历、能力不达标，面试的人力比较好，虽然说话很伤人，但是这样简单粗暴的方式却让我再一次认真审视自己，也参考了Ta给的几个建议，最后按照这个建议投简历，仍然不喜欢。现在想想，自己的方向早就在心中定了，按照别人给的建议走路当然会心不在焉，不得其法。

此阶段几点心得：
1、定位要准

心里要时刻清晰自己想要发展的方向，直线或者曲线救国都OK，前提是目标一定要明确，即使投三种不同的行业，只要它利于你最终实现职业目标都OK。海投无意义，你不会面，别人也不会要。

2、定位要稳

下定决心要进入的行业、拥有的职位，就要始终坚持。即使被多次打击，也要学会不断举例肯定自己，即使没有行业经验又怎样，学习+时间可以弥补这一切，但是如果因一时的打击而回归老本行或者放弃，只会更加痛苦。如果提前对自己有过客观的剖析，认可自己的能力以及相信自己未来可以达到预期，那就更应该坚持。

3、充分准备

这一点我一开始自己做的其实不好，但是被打击后开始认真准备，毕竟是转行，该了解的知识要了解，尽量多找些自己过去工作与新行业挂钩的地方，这样面试时很有用。有备无患，自信源于此。

4、平和心态

如果说这阶段最重要的一点就是，保持平和的心态。我在过程中会用三种方法保持好心态：

A、工作迟早会有的，好事多磨。

B、自信自信再自信，我知道我的能力、长处，他人的否定只是对我某方面的片面否定， 而非对我整个人的否定，我知道自己想要什么、能做什么。

C、生活积极向上，不让找工作充斥整个生活，控制好能控制的部分，每日定时投简历，然后做一切能让自己有控制感的事情，比如健身、敷面膜、和朋友吃饭、看书学习，这些真的很有用。

年末又是新一轮求职记，整理了一下具体的转行求职步骤进行分享。

文章以实用性为宗旨，最终目的是提供找工作路上的朋友一些帮助；

为方便阅读，我将从个人概况、转行定位、求职渠道及面试技巧四方面写，欢迎大家各取所需。

一、个人概况
楼主之前有过两年工作经验，都是跟地产挂钩的，包括商业地产和房地产的营销策划，可以说一直在地产行业混，而职位始终定位于营销策划岗。地产是个看起来很美的行业（当然了，大部分行业都是这样），当年选择进这个行业也是为了满足虚荣心，因为工资高，毕业第一年月薪6000，对于刚毕业的小孩子来说还是很有吸引力的，而当时选择做营销一是与大学专业挂钩，二是营销岗在地产行业还是属于核心岗位的。一开始楼主也很有干劲，但随着时间推移，发现工作内容始终很有局限，刚开始做什么都有成就感，到后来做什么都在应付。这种状态持续不久，我就开始考虑辞职的事，并且也发现自己确实不怎么喜欢这个行业。

折磨归折磨，痛苦归痛苦，如果你觉得哪里不舒服，那里一定有问题，任凭情绪发展没有用，必须得直面、分析、解决。

楼主仔细分析了一下想了想为什么想要离开这一行、这份工作，大概有以下几点原因：

1、缺乏热爱：当初找工作更多的是看工资，然后真正做起来时内心百般痛苦，倒也不是自己做不来，真的是觉得大部分时间都不开心，心已经告诉自己答案了；

2、画地为牢：营销策划说起来很好听，其实是个很虚的概念，大部分时间都在加班，但是实质上你的那些事真正对商场销售、楼盘销量带动了多少是很难计算出来的，一旦销售被带动，很大程度上功劳会归于营运、销售等其他部门。你在工作上忙活了很久也确实落地执行了，但效果如何真的很难量化，而我喜欢数字告诉我真相；

3、与个人规划方向有背：这点就得自己觉悟，总体而言，我更希望一份工作让我学到工作以外的知识，脱离了这份工作我依然能存活，是我定义工作、而不是工作证明我。

下定决心后，楼主就辞职了，当然当时也没有想清楚下一步怎么走，但是离开的原因想清楚就够了。

二、转行定位
楼主辞职后，因为个人原因，来北京发展。过完年的一个月，我开始陆陆续续投简历，但是成效不大，自己也没有很上心，觉得先投着再说。过了半个月，开始觉得不对了，因为自己一直是个自信找工作很容易的人，结果事态的发展却不是那么回事。我开始自己考虑各种原因，最重要的一点：没有定位，盲目投简历！

这一点说起来应该是最重要的，因为自己毕竟工作过两年，也知道经常跳槽对职业生涯的影响，所以找工作前实现思考一下职业规划很重要，职业规划的起始点就是职场定位。盲目的投简历一来不匹配的公司不会面试你，二来真面上了估计也不会去工作，何必费那无用功呢~

当时我对自己的定位过程大概是这样的：

1、网络广泛搜集个人感兴趣行业

2、搜查行业内外部信息：内部指行业论坛真人吐槽、外部指行业官方定义

3、找行业熟人问询，了解真实情况

4、仔细思考个人优劣势：可以想想在过去的工作中你做过哪件事让你最有成就感、最开心，为什么？别人经常表扬你哪一点？

综合以上，锁定一两个相关行业，重点研究其中哪个岗位与自己匹配度最高，最契合个人发展规划。

如果你有心仪的转行目标，那就最好了，我很幸运之前就对公关行业感兴趣，之前做地产营销策划时也发现自己更喜欢活动策划及执行这一块，看着一个项目在自己手上从无到有的感觉特别好。考虑转行公关基于以下几点：

公关公司属于乙方，服务于不同行业公司，在其中学到的能力是专业能力，在对企业做传播时所用的信息监测等手段都是专业手段，切实可以看到成效；

虽然属于转行，但我个人之前做营销时也接触过媒体、活动策划及信息传播这方面的内容，所以不属于从零开始，工作上还是有些延续性的；

我可以看到未来自己需要学习哪些知识与技能、可以得到怎样的发展，也可以看到它对我梦想的支持性之大。（关于五年后成为什么样）

综上，我确定了公关这个行业，并且在对岗位做过细分后，决定做媒介或客户这两个方面的岗位，一来它们基本属于公关行业的核心岗位（看需求量、招聘量），二来可以拥有自己的资源。

同时因为转行，我希望自己一开始就进一家大规模的企业，在一个比较完善的系统下得到锻炼，因为个人一直觉得，职涯的前三年在大公司锻炼，学习专业知识，中后期可以到小公司锻炼管理能力。

在这个认知的前提下，我将简历投递的范围限定在公关行业世界排名前十及国内前十的公司。

三、求职渠道
前面的定位完成后，就是写简历、投递简历，简历在某方面体现了个人综合素质，这一块网上有很多资料，我当时是中英文都准备了，并且找国外朋友帮忙修改，同时自己也改了四五遍才最终定稿。

Tips：写简历最重要的一点就是要和求职岗位匹配度高，有个小窍门，可以搜索自己心仪的岗位，看其职位要求及岗位职责，一条条罗列下来，再把自己过去工作中符合的经验与之匹配写出来，匹配度越高越好。

关于投递简历的渠道，本着不走寻常路的原则，大概有以下几条：

1、官网：推荐最优先使用。直接搜索公司官网，然后找到招聘类的栏目，按所列邮箱投递个人简历。我后来收到offer就是在官网投递得到的回复。

2、社交网站：第二推荐，要善用社交网站，比如新浪微博、豆瓣等。可以搜索企业的官方招聘微博，里面会发布许多有效职位，豆瓣招聘小组里面也有许多好公司及职位。针对性更强、能缩短求职许多道繁琐的程序。

3、招聘网站：智联、大街网等，不多说了。我觉得用处不大，因为里面的很多职位都是长期挂着的。

总之，这阶段你主要考虑的问题就是如何让企业更快地看到你的简历并且想要面试你，如果只用常规求职途径当然会麻烦很多、效果也好不到哪里，因为人才太多了。如果这个阶段你可以多动动脑子、多找资源，会容易很多，例如外企有内部推荐制度，这一点就可以充分去利用。

四、面试技巧
这一部分其实最没什么好写的，因为面试问题总共就那么几个，当然由于要转行，面试官肯定会问及为何选择这个行业，我当时在准备英文面试时发现标准答案遵循的原则永远是：

无论问题是什么，回答的内容永远围绕我如何如何与这个岗位匹配、我过去的工作经验能为公司提供什么。

实用性的语句比类似“我真的很爱这份工作”巴拉巴拉等等空虚的句子更显得人的理性，面试官会相信你是在理性状态下选择这份职业、而他们选择你也完全是出于理性考虑。

五、友情提示
找工作期间，人很容易在孤独中自怨自艾，但是工作肯定会有的，只是早晚的事，给几点友情提示，绝对有用：

1、保持通讯顺畅：手机在8:30-18:00保持开机，尤其是爱睡懒觉的小伙伴，一定在早上要正常开机。下午三四点的时候要注意手机有无来电。

2、正常饮食起居：早睡早起、敷面膜、锻炼、正常吃饭，自我控制、保持生活节奏很重要，这个时候能够自我控制会在一定程度上增强自信。

3、终极法宝：自信自信自信，不要因为时间久找不到工作就随便找一家下嫁（这里指特别低于心里预期的工作），要相信总会找到工作，相信自己的能力（理性认知的基础上）。找工作只是累这么一阵子，换来的是以后长久的确定幸福。

OK，终于写完了，这阶段我也低沉过、怀疑过，但始终很相信自己，希望你也是，祝愿大家都能找到满意的工作！
