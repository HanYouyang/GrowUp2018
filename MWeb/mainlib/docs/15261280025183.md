# 写作套路
通行五段法：
开头：复述标题并联系自己所想所写的内容
结尾：复述之前自己联系的内容，推而广之；有争议的要说明一下怎么做太肤浅/简单而不应该
中间三段：运用论证方法论证自己的观点或者想法的合理性
连缀：在每个部分的书写上面都要有自己的内在关联要写出来

02评析一句话：
可以对这句话不断琢磨改进，让这句话形成合理空间
或者举出反例在不同点证明错误


135-66-124-107极端二选一：
认同其中一个的情况下：给出一个自己现在的理由，并直接说明优点
而后选择特殊情况说明不一定适用/对于不同意的理论可以直接说明情况成立的时候荒谬的情况
最后结尾给出折中的解决方案方案，重复强调优点
不选择任何一个的情况下：写成三段论，自己的观点是折衷选择的结果
说明不能极端选择后说明两者各自的情况优点缺点


“几选一”：
选中一种自己说明的方法
事物有自己运行的方式：
了解各个成分
根据情况调整组成方式
寻找整体改进空间
也可以指出其他方法或者观点的错误

给出一个想法解决问题：
开头加上自己观察到的事实，自己给出的想法，这样做的原因一并写出来，也就是下面几个段的开头标题。
中间1：整体现状+细节现状+双向困境+解决方法+效果
中间2：方法整体缓解+细节改善+诉诸方法背后的客观规律+规律应用到实际中的情况+进一步说明双向好处延伸过去
